# Simulador-envios-POO
Proyecto primer corte POO -Simulación de envíos Java por Aaron Steven Hernández y Juliana Gutiérrez


# Descripción
Este proyecto consiste en el desarrollo de un sistema en Java que simula el cálculo del costo y el tiempo de envío de mercancías en una empresa de tecnología logística. Se aplican principios fundamentales de la programación orientada a objetos, como clases abstractas, herencia y polimorfismo, para modelar diferentes tipos de transporte.


# Objetivos
- Aplicar los principios de la programación orientada a objetos.
- Implementar clases abstractas y herencia.
- Definir métodos abstractos y su sobrescritura en clases concretas.
- Simular el cálculo de costo y tiempo de envío.
- Comparar diferentes medios de transporte.


# Funcionalidades
- Cálculo del costo de envío según distancia y carga.
- Estimación del tiempo de entrega con base en la velocidad.
- Validación de la capacidad máxima de carga.
- Generación de un reporte comparativo entre vehículos.

# Conceptos aplicados
- Clases abstractas
- Herencia
- Polimorfismo

# Tipos de vehículos
TERRESTRES
- Auto
- Moto
- Camión
- Bicicleta

AÉREOS
- Dron
- Avión


# Ejecución del programa
1. Clonar el repositorio:
git clone <URL_DEL_REPOSITORIO>

2. Entrar a la carpeta
cd Simulador-envíos-POO

3. Compilar
javac propoo/*.java

4. Ejecutar
java propoo.proPOO


# Ejemplo de uso
Destino: Antioquia
Peso: 50 kg
El sistema mostrará:
* Vehículos disponibles
* Costo de envío
* Tiempo estimado
